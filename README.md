# Silent Performance Improvement Example

This example demonstrates the concept of 'silent improvements' by showcasing a performance optimization. It compares two versions of a data processing task: one less optimized using basic string concatenation, and one with a subtle code change using f-strings for more efficient concatenation. The output is functionally identical, but the optimized version executes significantly faster, illustrating how users experience a better product without noticing specific changes to features or UI.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
